Abstract

The Blind Trust Problem in Video-LLMs

Video reasoning models implicitly assume every input frame is equally reliable. Under realistic perturbations — motion blur, glare, or occlusion — frontier Video-LLMs lose 15–30 accuracy points on real-world embodied benchmarks while their self-reported confidence remains unchanged. We call this the Blind Trust Problem.

We propose Robust-TO, an agentic video understanding framework that explicitly integrates per-frame trustworthiness into every stage of reasoning. Robust-TO organizes heterogeneous visual perception tools under a unified (result, confidence) interface, where each tool returns a calibrated confidence score that multiplicatively couples its intrinsic certainty with a parameter-free disturbance estimate of its input frames.

On two corruption-robustness video benchmarks spanning eight tasks, Robust-TO improves over the strongest open-source baseline by +10.6 points on average, while introducing less than 5% latency overhead on clean inputs. It achieves the highest clean accuracy and the smallest clean-to-corrupted accuracy drop among all compared methods.

Method

Three Coordinated Mechanisms

Robust-TO is built on a unified (result, confidence) tool interface. Each perception tool returns a calibrated confidence score coupling its intrinsic certainty with a parameter-free disturbance estimate of its input frames — enabling three coordinated mechanisms:

🔍

Disturbance-Aware Evidence Acquisition

Frames are jointly ranked by reliability × query relevance. Sub-queries are decomposed and routed to the tool best matched to the dominant corruption type (blur → captioning; occlusion → action recognition). No retraining is needed when adding new tools.

⚖️

Confidence-Weighted Evidence Synthesis

Facts are grouped into three reliability tiers (HIGH / MEDIUM / LOW). The answer is derived primarily from high-reliability facts; contradictory medium-tier facts are discarded; low-tier facts are used only when no stronger evidence exists, with explicit uncertainty flagging.

🎯

Confidence–Cost GRPO Reward

A training reward that jointly optimizes correctness, evidence reliability, and efficiency. A sub-query efficiency term with a frozen N* estimator prevents reward gaming while encouraging minimal, well-targeted tool calls.

🔌

Plug-and-Play Tool Library

Five default tools — detect_objects, caption_frame, track_temporal, recognize_action, read_text — each exposed through the unified interface. New tools register by declaring semantic type and cost only.

Experiments

Main Results

Evaluated on UrbanVideo-Bench (outdoor embodied reasoning: Landmark Position, Counterfactual, Progress Evaluation, Action Generation) and VSI-Bench (indoor spatial intelligence: Relative Distance, Relative Direction, Route Planning, Appearance Order). Corrupted variants generated by the RoVA-V1 video masker (Motion Blur, Gaussian Noise, Glare, Occlusion, Low-Light).
